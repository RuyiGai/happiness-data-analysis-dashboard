\# Happiness Data Analysis Dashboard



\## Overview

This project analyzes the World Happiness Report dataset and presents insights through an interactive data visualization dashboard built in Python.



\## Tools

\- Python

\- Pandas

\- Plotly

\- Jupyter Notebook



\## Data Source

The datasets used in this project are publicly available:

\- World Happiness Report 2024

\- UN Region Categories dataset



The data was downloaded from public online sources and used for educational purposes.



\## Description

The project includes data cleaning, exploratory data analysis (EDA), and interactive visualization to explore country-level and temporal trends in happiness indicators.
